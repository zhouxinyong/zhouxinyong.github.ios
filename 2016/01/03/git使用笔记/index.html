git使用笔记

  • git
  • 笔记

工作和学习笔记，便于以后快速查找

git常用命令速查表

序号 代码 操作说明
1 git clone 克隆远程版本库
2 git init 初始化本地版本库
3 git status 查看当前版本库状态
4 git diff 查看变更的内容
5 git add . 跟踪所有改动过的文件
6 git add 跟踪指定的文件
7 git mv 文件改名
8 git rm 删除指定文件
9 git rm –cached 停止跟踪文件但不删除
10 git commit -m “comment on commit” 提交所有文件
11 git commit –amend 修改最后一次提交
12 git log 查看历史提交记录
13 git log -p 查看指定文件的提交历史
14 git blame 以列表方式查看指定文件的提交历史
15 git reset –hard HEAD 撤销工作目录中所有未提交文件的修改内容
16 git checkout HEAD 撤销指定未提交文件的修改内容
17 git revert 撤销指定的提交
18 git branch 显示所有本地分支
19 git checkout 切换到指定分支或标签名对应的分支
20 git branch 创建本地分支
21 git branch -d 删除本地分支
22 git tag 显示本地所有标签
23 git tag 基于最新提交创建标签
24 git tag -d 删除标签
25 git merge 合并指定分支到当前分支
26 git rebase 衍和指定分支到当前分支
27 git remote -v 查看远程版本库信息
28 git remote show 查看指定远程版本库信息
29 git remote add 添加远程版本库
30 git fetch 从远程库获取版本
31 git pull 下载代码及快速合并
32 git push 上传代码及快速合并
33 git push : 删除远程分支或标签
34 git push –tags 删除所有标签
35 git log –pretty=format:”%h %s” –graph 展示每个提交所在的分支及其分化衍合情况

参考资料

  1. git book : https://git-scm.com/book/zh/v1/
